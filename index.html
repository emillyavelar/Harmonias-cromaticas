O Círculo Cromático

O círculo cromático é uma representação visual das cores organizadas em um espectro contínuo. Ele ajuda a entender as relações entre as cores e como elas podem ser combinadas para criar harmonia visual.

Harmonias Cromáticas

As harmonias cromáticas são combinações de cores que criam efeitos visuais equilibrados. As principais são:

  • Monocromática
  • Complementar
  • Análoga
  • Triádica
  • Tetrádica
  • Complementar Dividida

Como Escolher as Cores do Site

Para um design coerente, cada elemento deve ter uma cor pensada estrategicamente:

  • Cabeçalhos: Cor mais chamativa para destaque.
  • Parágrafos: Tons suaves para melhor leitura.
  • Bordas: Cor de contraste para separação visual.
  • Listas: Usar marcadores em tons harmônicos.
  • Links: Destaque sem comprometer a legibilidade.
